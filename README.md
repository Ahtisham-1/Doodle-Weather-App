# Doodle Weather App

A fun, boxy, and colorful weather application built with Vanilla JavaScript and the OpenWeatherMap API.

## Features
- **Real-time Weather:** Get temperature and descriptions for any city.
- **Loading State:** A wobbly animation for those "thinking" moments.
- **Doodle UI:** A hand-drawn, high-contrast aesthetic.
- **Error Handling:** Let's you know when a city doesn't exist.

##  Built With
- HTML5 & CSS3
- JavaScript (Async/Await Fetch)
- OpenWeatherMap API

##  How to Run
1. Clone this repo.
2. Open `index.html` in your browser.
3. Enter a city name and enjoy!
